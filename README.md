Building a website for Loya Eye Hospital. This website will display all the details, treatments, surgeries, diagnosis provided by the hospital. It will also display the awards, and achievements of doctors and testimonials of patients.


# Loya Eye Hospital Website

## Tech Stack

- **Backend:** Python Django 4.2
- **DB:** PostgreSQL
- **Frontend:** Django Templates + Bootstrap 5 (CDN)
- **CMS:** Django Admin
